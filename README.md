# 🚀 AI Resume & Interview Coach (Powered by Gemini AI)

An AI-powered web application that analyzes resumes and generates interview questions using Google Gemini API.

---

## 💡 Project Overview

This project helps students and job seekers improve their resumes and prepare for interviews using AI.

Inspired by real-world AI career tools, this app provides instant feedback, ATS scoring, and interview preparation in one place.

---

## 🔥 Features

- 📄 Resume Analysis using AI
- 📊 ATS Score generation
- ✅ Strengths & Weakness detection
- 💡 Smart improvement suggestions
- 🎯 AI-generated interview questions
- ⚡ Real-time AI response using Gemini API

---

## 🧠 How It Works

1. User pastes resume text  
2. App sends data to Gemini API  
3. AI analyzes resume  
4. Results displayed instantly  

---

## 🛠️ Tech Stack

- HTML, CSS, JavaScript  
- Google Gemini API  
- Vercel (Frontend Deployment)  
- Google Cloud Run
